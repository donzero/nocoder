设置全局目录变量

进入 shell 中，每次输入 cd /usr/java/jdk1.6.0_43 或更多级目录让人头疼，可以设置全局变量，每次进入可以如下使用：

  1. 编辑 profile 文件
    1. 用 vi 或 vim 编辑 /etc/profile 文件（root 用户才有可写权限, 也可以 chmod u+w /etc/profile 给用户添加可写权限）: vim /etc/profile ；
    2. 按键盘 i 进入编辑模式，在文件末尾 unset i 前添加 export JAVA_HOME=/usr/java/jdk1.6.0_43，按键盘 Esc，输入 :wq! 保存文件；
  2. Source 让设置立即生效
    1. 想让配置立刻生效，而不用重新登录用 source 命令：source /etc/profile
  3. 你可以使用 cd $JAVA_HOME 快速进入 jdk 目录了，其他脚本中也可以使用该变量；

sudo

  1. sudo -l 查看sudo权限
  2. sudo -u username /bin/mkdir logs 新建 logs 文件夹
